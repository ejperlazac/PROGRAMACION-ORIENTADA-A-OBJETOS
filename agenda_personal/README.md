# Agenda Personal en Python

Aplicación gráfica desarrollada con Tkinter que permite gestionar eventos personales.

## Funcionalidades
- Agregar eventos con fecha, hora y descripción
- Validación de datos
- Eliminación de eventos con confirmación
- Visualización en tabla (TreeView)
- Uso de DatePicker

## Tecnologías utilizadas
- Python
- Tkinter
- ttk
- tkcalendar

## Instalación
pip install tkcalendar

## Ejecución
py agenda.py

## Características adicionales
- Validación de formato de hora (HH:MM)
- Mensajes de confirmación y éxito
- Scroll en la tabla
