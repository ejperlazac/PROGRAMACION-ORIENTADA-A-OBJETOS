# Aplicación GUI Básica

Aplicación desarrollada en Python utilizando la librería Tkinter para implementar una interfaz gráfica de usuario.

## Funcionalidades

- Ingreso de datos mediante campo de texto
- Agregar elementos a una lista
- Eliminar elementos seleccionados
- Limpiar la lista completa

## Tecnologías

- Python
- Tkinter
- Git

## Ejecución

```bash
python main.py
