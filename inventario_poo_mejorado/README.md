# Sistema de Gestión de Inventarios Mejorado

## Descripción
Este proyecto corresponde a la mejora del sistema de gestión de inventarios desarrollado previamente en la asignatura de Programación Orientada a Objetos.

La mejora implementa almacenamiento persistente en archivos de texto y manejo de excepciones durante la lectura y escritura de datos.

## Objetivo
- Almacenamiento automático del inventario en un archivo de texto.
- Recuperación automática de datos al iniciar el programa.
- Manejo de excepciones como FileNotFoundError y PermissionError.
- Notificación al usuario sobre el éxito o fallo de las operaciones.

## Estructura del Proyecto
inventario_poo_mejorado/
- producto.py
- inventario.py
- main.py
- inventario.txt
- README.md

## Funcionalidades
- Agregar productos
- Eliminar productos
- Actualizar productos
- Mostrar inventario
- Guardado automático en archivo
- Carga automática al iniciar

## Manejo de Excepciones
El sistema maneja:
- FileNotFoundError
- PermissionError
- Excepciones generales

## Ejecución
Desde la terminal:

py main.py
