# Ejemplos del Mundo Real y Programación Orientada a Objetos

## Descripción
Este proyecto presenta un ejemplo práctico de Programación Orientada a Objetos mediante la simulación de un sistema de reservas de hotel. El programa modela una situación del mundo real utilizando clases, atributos y métodos en Python.

## Clases implementadas
Cliente  
Habitacion  
Reserva  

## Principios de POO aplicados
Encapsulación  
Modularidad  
Interacción entre objetos
