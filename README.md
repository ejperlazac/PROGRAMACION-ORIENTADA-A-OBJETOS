# Aplicación GUI de Gestión de Tareas

## Descripción
Esta aplicación permite gestionar tareas mediante una interfaz gráfica desarrollada con Tkinter. El usuario puede añadir, completar y eliminar tareas utilizando tanto botones como atajos de teclado.

## Funcionalidades
- Añadir tareas mediante botón o tecla Enter.
- Marcar tareas como completadas con botón o tecla "C".
- Eliminar tareas con botón o tecla "Delete".
- Cerrar la aplicación con tecla "Escape".

## Interfaz
- Campo de entrada para nuevas tareas.
- Lista visual de tareas.
- Botones de interacción.

## Características adicionales
- Feedback visual:
  - ✔ Tareas completadas (color verde)
  - ✗ Tareas pendientes (color negro)

## Tecnologías utilizadas
- Python
- Tkinter
