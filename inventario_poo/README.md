# Dashboard de Gestión de Tareas – Programación Orientada a Objetos

## Descripción general
Este proyecto corresponde a la adaptación del archivo Dashboard.py como parte de la asignatura Programación Orientada a Objetos. El objetivo es implementar un dashboard interactivo en Python que permita gestionar tareas académicas aplicando principios de programación orientada a objetos.

El sistema se ejecuta en consola y presenta un menú interactivo para facilitar la gestión de tareas.

## Objetivo
Desarrollar un dashboard funcional que permita agregar, listar y completar tareas, promoviendo la organización del código, la modularidad y el uso de buenas prácticas en Python.

## Funcionalidades
El archivo Dashboard.py permite:
- Mostrar un menú principal interactivo
- Agregar tareas POO
- Listar tareas registradas
- Marcar tareas como completadas
- Salir del sistema de forma controlada

## Enfoque de Programación Orientada a Objetos
El desarrollo del dashboard aplica principios de POO mediante:
- Organización modular del código
- Separación de responsabilidades
- Uso de funciones bien definidas
- Código legible y mantenible

## Documentación del código
El archivo Dashboard.py contiene comentarios descriptivos que explican el propósito de cada función y bloque lógico, facilitando su comprensión y mantenimiento.

## Ejecución del programa
Para ejecutar el programa, utilizar el siguiente comando desde la terminal:

python Dashboard.py
