# Sistema de Gestión de Inventarios – Programación Orientada a Objetos

## Descripción general

Este proyecto consiste en el desarrollo de un sistema de gestión de inventarios implementado en Python, aplicando principios de Programación Orientada a Objetos.

El sistema permite registrar, almacenar y administrar productos mediante una estructura organizada en clases.

La aplicación se ejecuta en consola e incorpora un menú interactivo.

---

## Objetivo

El sistema permite:

- Agregar productos
- Mostrar productos registrados
- Buscar productos por nombre
- Gestionar información mediante clases
